## 🎯 Objective

Design and implement a desktop-based Online Store application that supports:
- Role-based login for Admins and Customers
- Admin dashboard with store management options
- Customer signup and login with ID validation
- Customer dashboard to browse and interact with the store
- Scene navigation across all views using JavaFX

---

## 👤 My Responsibilities

This was a group project. My assigned pages and responsibilities:

| Page | Frontend (FXML/UI) | Backend (Controller/Logic) |
|------|-------------------|---------------------------|
| Login View | Login UI with Admin & Customer fields | Login logic, scene switching, Signup button |
| Admin Dashboard | View, Add, Back buttons | Back button controller |
| Customer Dashboard | View, Buy, Back buttons | Back button controller |
| Signup Dashboard | Signup form with ID field | Signup logic, Add button |

---

## ✨ Features

- Role-based login — separate flow for Admin and Customer
- Admin login with username validation
- Customer signup — register with a unique ID
- Customer login — validated against registered IDs
- Scene switching between all views using JavaFX SceneBuilder

## 🚀 How to Run

1. Make sure **Java JDK** and **JavaFX SDK** are installed
2. Open the project in your IDE (IntelliJ / Eclipse)
3. Add JavaFX libraries to the module path
4. Run `Main.java`
